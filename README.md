# 야간 주행 스마트 시스템
SeSAC 임베디드 프로젝트 5조 - 야간주행 스마트시스템

야간 환경에서의 자동차 시스템을 임베디드 시스템으로 구현한 프로젝트입니다.

##:pushpin: 프로젝트 구성

3가지 내부 시스템이 있으며, 폴더별로 나누어져 있습니다.
1. 문콕 방지 시스템 (DOOR)
2. 자동 상향등 시스템 (AHB)
3. BMS 모니터링 시스템 (BMS)
![Image](https://github.com/user-attachments/assets/33030bc7-2c3d-44d5-bc6f-1b7b635ce7b5)
